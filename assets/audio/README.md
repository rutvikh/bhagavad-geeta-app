# Audio Files

Place verse recitation MP3 files here.

Directory structure:
  ch01/verse_01_01.mp3
  ch01/verse_01_02.mp3
  ...
  ch02/verse_02_47.mp3
  ...

Requirements:
- Format: MP3, 64kbps mono (sufficient for chanting audio)
- Duration: typically 20–60 seconds per verse
- Size: ~0.5MB per verse average
- Total (700 verses): ~350MB

Sources for audio:
- Recorded by yourself or a traditional chanter
- Sanskrit recitation apps / Gita audio repositories
- Commissioned from a Sanskrit chanting institution

Audio files are optional — the player gracefully shows "Audio not available"
when a file is missing.
